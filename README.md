# NodeColor
A color palette for nodejs

|  Color | HEX  |
| ------------ | ------------ |
|  apricot  |   #E67E30 |
|  mahogany  |   #88421D |
|  aquamarine  |   #79F8F8 |
|  chestnut  |   #A76726 |
|  almond  |  #82C46C |
|  amaranth  |   #91283B |

Usage:

```javascript
const color = require("nodecolor")

/* For exemple */
console.log(color.mahogany)
///=> #88421D
```
